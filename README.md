# ATM Simulation System using Java
## Project Overview
This is a simple console-based ATM Simulation System developed using Java. The project performs basic banking operations through a menu-driven interface.

## Features
- Check Balance
- Deposit Money
- Withdraw Money
- Exit System

## Technologies Used
- Java
- OOP Concepts
- Scanner Class
- Conditional Statements
- Loops

## How to Run

### Compile
```bash
javac ATMSystem.java
